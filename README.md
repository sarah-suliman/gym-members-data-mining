Gym Members Data Mining

This project analyzes a gym member exercise dataset using data mining techniques to:

• Discover fitness profiles using K-Means clustering
• Predict calories burned with regression models
• Predict member experience level using classification
• Visualize relationships between health and workout variables

Tools:
• Python
• Pandas
• Scikit-learn
• Matplotlib
• Seaborn

Dataset:
Gym Members Exercise Dataset (Kaggle)
